# ENVI Music — Artist Portfolio

A personal music portfolio and player for ENVI, built as a single-page React application. Features a custom audio player, media showcase, and music resources page.

**Live site:** [envimusic.netlify.app](https://envimusic.netlify.app)

---

## Tech Stack

- **Framework:** React (Vite)
- **Styling:** CSS
- **CDN:** Cloudinary (audio and image delivery)
- **Deployment:** Netlify

---

## Features

- Custom audio player with track queue across multiple albums
- Music page organized by release — RETURN EP, Heds Tapes, Ghost Love, Previous Releases
- Media page with YouTube embeds (RETURN EP animations) and a 31-Day Blender 3D Challenge showcase
- Music Resources page with downloadable/purchasable assets (Vital Presets via Gumroad)

---

## Pages

| Route        | Description                               |
| ------------ | ----------------------------------------- |
| `/`          | Home / landing                            |
| `/music`     | Audio player with full discography        |
| `/media`     | YouTube embeds + Blender challenge tweets |
| `/resources` | Downloadable music resources              |
